# Türkçe Authenticator Projesi

Türkçe Authenticator, kullanıcıların uygulamalarında veya web sitelerinde iki faktörlü kimlik doğrulama (2FA) kullanmalarını sağlayan bir projedir.

## Özellikler

- İki faktörlü kimlik doğrulama desteği
- Kullanıcı dostu arayüz
- Güçlü güvenlik

## Kurulum

1. Projeyi klonlayın:
   ```bash
   git clone https://github.com/krkmz91g-cmd/turkce-authenticator.git
   ```
2. Gerekli bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
3. Uygulamayı başlatın:
   ```bash
   npm start
   ```

## Kullanım

Uygulama başlatıldıktan sonra, kullanıcıların kimlik doğrulama işlemlerini gerçekleştirebilecekleri bir arayüz göreceksiniz.
